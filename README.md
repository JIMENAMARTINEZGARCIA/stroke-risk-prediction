# stroke-risk-prediction

## 📌 Overview
This project analyzes clinical data to predict the risk of stroke using machine learning techniques.

## 🎯 Objectives
- Perform exploratory data analysis (EDA)
- Clean and preprocess healthcare data
- Handle class imbalance using SMOTE
- Train a classification model

## 🛠️ Tools & Technologies
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib, Seaborn

## 📊 Results
The model shows that variables such as age, hypertension, and BMI are significant predictors of stroke risk.

## 🧠 Medical Insight
This project highlights how data analysis can support early detection and prevention strategies in healthcare.
